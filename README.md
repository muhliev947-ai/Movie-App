🎬 Movie App  Интерактивное приложение для поиска фильмов с использованием внешнего API.
Фокус на чистом интерфейсе, удобной навигации и продуманной логике фильтрации. 
✨ Основные возможности      
🔍 Поиск фильмов по названию     
🎞️ Просмотр постеров, рейтингов и описаний    
🏷️ Фильтрация по жанрам     
⭐ Отображение рейтинга и ключевой информации    
⚡ Быстрый отклик и оптимизация запросов    
📱 Полностью адаптивный интерфейс  
🛠️ Технологии     
Frontend: React    
Язык: JavaScript    
UI: Tailwind CSS / Custom CSS   
API: TMDB / OMDb (в зависимости от твоего проекта)     
Инструменты: Git, VS Code, Linux  
🎯 Цели проекта     
Практика работы с внешними API     
Управление состоянием в React     
Создание удобного UI/UX    
Работа с асинхронными запросами
